# University Management System
A Java-based desktop application built with Swing and MySQL that allows administrators to manage students, faculty, marks, fee records, and leave applications from a single unified interface. The system demonstrates core Object-Oriented Programming principles including encapsulation, abstraction, polymorphism, and interface implementation across all modules.

## Purpose
Universities deal with large volumes of student and faculty data that are difficult to manage manually. This system solves that problem by providing a centralized desktop application where an administrator can:

- Register and update student and faculty records
- Enter and view semester-wise exam marks
- Apply and track student and teacher leave requests
- Calculate and record semester fee payments
- View detailed records for students, faculty, exams, and leaves



**Users:** University administrators and office staff.
